Bulk page creation for setting up quickly your website. Intuitive and easy to use.

== Description ==

This plugin is mainly intended for developers for the basic configuration of a WordPress sites.

Its purpose is to provide a simple interface to easily create a page tree structure with also the ability to create a menu from theses pages. It's only used on the WordPress back-end.

Just go to `Pages` => `Add multiple pages` to use the plugin.

== Installation ==

Install and activate the plugin from the 'Plugins' menu in WordPress as usual.
